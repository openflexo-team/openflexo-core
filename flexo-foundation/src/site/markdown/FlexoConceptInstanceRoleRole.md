## Usage

```java
[visibility] [cardinality] FlexoConceptInstanceRole <identifier> with FlexoConceptInstanceRoleRole();
```

or

```java
[visibility] [cardinality] FlexoConceptInstanceRole <identifier> with FML::FlexoConceptInstanceRoleRole();
```

where

- `visibility` is default (unspecified), 'public', 'protected' or 'private'
- `cardinality` is \[0,1\] (unspecified), \[0,\*\] or \[1,\*\]
- \<identifier\> is the name of declared model slot variable
