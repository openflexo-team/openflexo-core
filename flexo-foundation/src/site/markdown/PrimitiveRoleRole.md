## Usage

```java
[visibility] [cardinality] PrimitiveRole <identifier> with PrimitiveRoleRole();
```

or

```java
[visibility] [cardinality] PrimitiveRole <identifier> with FML::PrimitiveRoleRole();
```

where

- `visibility` is default (unspecified), 'public', 'protected' or 'private'
- `cardinality` is \[0,1\] (unspecified), \[0,\*\] or \[1,\*\]
- \<identifier\> is the name of declared model slot variable
