## Usage

```java
[visibility] [cardinality] ActionScheme <identifier> with ActionSchemeRole();
```

or

```java
[visibility] [cardinality] ActionScheme <identifier> with FML::ActionSchemeRole();
```

where

- `visibility` is default (unspecified), 'public', 'protected' or 'private'
- `cardinality` is \[0,1\] (unspecified), \[0,\*\] or \[1,\*\]
- \<identifier\> is the name of declared model slot variable
