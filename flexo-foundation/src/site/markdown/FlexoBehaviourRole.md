## Usage

```java
[visibility] [cardinality] FlexoBehaviour <identifier> with FlexoBehaviourRole();
```

or

```java
[visibility] [cardinality] FlexoBehaviour <identifier> with FML::FlexoBehaviourRole();
```

where

- `visibility` is default (unspecified), 'public', 'protected' or 'private'
- `cardinality` is \[0,1\] (unspecified), \[0,\*\] or \[1,\*\]
- \<identifier\> is the name of declared model slot variable
