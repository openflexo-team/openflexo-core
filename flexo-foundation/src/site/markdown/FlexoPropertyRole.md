## Usage

```java
[visibility] [cardinality] FlexoProperty <identifier> with FlexoPropertyRole();
```

or

```java
[visibility] [cardinality] FlexoProperty <identifier> with FML::FlexoPropertyRole();
```

where

- `visibility` is default (unspecified), 'public', 'protected' or 'private'
- `cardinality` is \[0,1\] (unspecified), \[0,\*\] or \[1,\*\]
- \<identifier\> is the name of declared model slot variable
