## Usage

```java
[visibility] [cardinality] FlexoConcept <identifier> with FlexoConceptRole();
```

or

```java
[visibility] [cardinality] FlexoConcept <identifier> with FML::FlexoConceptRole();
```

where

- `visibility` is default (unspecified), 'public', 'protected' or 'private'
- `cardinality` is \[0,1\] (unspecified), \[0,\*\] or \[1,\*\]
- \<identifier\> is the name of declared model slot variable
