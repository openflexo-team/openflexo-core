## Usage

```java
[visibility] [cardinality] FlexoConceptInstance <identifier> with ConceptInstance([options]);
```

or

```java
[visibility] [cardinality] FlexoConceptInstance <identifier> with FML@RT::ConceptInstance([options]);
```

where

- `visibility` is default (unspecified), 'public', 'protected' or 'private'
- `cardinality` is \[0,1\] (unspecified), \[0,\*\] or \[1,\*\]
- \<identifier\> is the name of declared model slot variable
